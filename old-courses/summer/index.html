Message from Mr. Smith about coding over the summer

I am excited to be able to offer four computer science classes this coming year, and I'm looking forward to seeing you in the fall!

Doing some coding over the summer is a great way to keep your brain active and prepare you for a successful year ahead. To be clear there is no required summer assignment. But if you are interested, below are some things you could do over the summer if you want to stay fresh and challenge yourself.

Of course you could also just try to beat A Dark Room or Candy Box 2.

-Mr. Smith

for students entering
Computer Programming

for students entering
AP Computer Principles

Get to know HTML and CSS by doing the first four lessons in Khan Academy's "Making webpages" course (starting with "Intro to HTML" up through and including "CSS Text Properties").
Here's the link.

Also: stretch yourself a bit in Python by exploring this online games book or this online cypher book. You could set up Python on your own computer, or build+run the code in repl.it (just make sure you create the correct type of repl: py3 or py or pygame).

Also, this is a good online book, and you can write+run python right on the webpages.
