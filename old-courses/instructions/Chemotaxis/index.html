Intro

Chemotaxis is when an organism moves toward or away from a chemical stimulus like food. Biologists write software using random walks to model and study how very small organisms like bacteria search for food. In this assignment you will create a simulation of organisms around an environment using a random walk while also responding to a stimulus. Your simulation shouldn't look like any other. Have fun and be creative.

Step 1: One Bacteria

Use the reference slides and practice worksheets to create a sketch with one Bacteria object which moves in a random walk around the screen. You might find it helpful to reference the Chemotaxis slides #1-28, see link in Materials section of the class page.

Step 2: Array of Bacteria

Use the reference slides and practice worksheets to modify your sketch so that it uses an Array. If you do this modification correctly then your sketch will show a group of Bacteria, each moving randomly around the screen.

Step 3: Bias

Update your code so that the random walk of the Bacteria are biased towards or away from a particular point on the screen. You decide what this point should represent. You may draw something like food or danger at this location, or just let it be the mouseX and mouseY.

Use a structure like this in the move method of the Bacteria class definition: 

                if (x < mouseX) {   // check if I'm to the left of the mouse
                  x = x + (int)(Math.random() * 7) - 2;  // more likely to get a positive number (ie. move to right)
                } else {  // I'm to the right of the mouse
                  x = x + (int)(Math.random() * 7) - 5; // more likely to get a negative number (ie. move to the left)
                }
and do something similar for the y variable of the class.

Step 4: U-Pick Creative Element

This step is open-ended and depends on your creativity. You need to add one additional element to your simulation. You decide what it is -- choose something that you find interesting and acheivable. The following are some examples you might choose from:

  • use the mouse buttons to adjust the bias towards or away from the mouse
  • modify your class so the bacteria have a finite lifespan and can reproduce
  • simulate other organisms besides bacteria and even have multiple types of organisms like a predator and prey
  • gradually change a Bacteria's color based on its position
  • ... or something else you come up with ...

Definition of Done

You are done when:

  • You have a runnable sketch on github, which is viewable at http://your-username.github.io/Chemotaxis.
  • You have a class to represent a bacteria or other organism, the following members:
    • at least 2 ints (to hold the x and y coordinates of the bacteria position)
    • a constructor which initializes the 2 ints
    • void move() which moves the bacteria in a random walk pattern
    • void show() which draws the bacteria using the three member variables
  • You use an array to store many instances of the Bacteria.
  • Your Bacteria has some sort of "biased random walk" towards some stimulus like a piece of food or the mouse.
  • Your program has one additional creative element, as per Step 4 above.
  • Each part of your code has comments so someone else can understand it.

After this you must fill out a 'Submit a Coding Project' form and put it in my IN box in Room 7.
