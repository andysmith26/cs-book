Goals

  • create a blog
  • add a post
  • publish it online

Github Set-Up

To do this assignment you'll need a github account. If you don't already have one, follow the instructions linked here.

Make sure you're signed in to github with your account, then fork the FirstBlog repo. (see How to Fork a Repo linked here)

Local Set-Up

Go to your copy of the lab repo. Click the green button 'Clone or Download', then click 'Download ZIP'. Extract the file to your Desktop (or another place you like to keep files).

In the new (extracted) folder called FirstBlog, open the index.html file. Make sure you see a webpage with moving clouds and some sample text.

First Post

Launch Atom and open the same index.html file in Atom. Replace the Blog Title with your own title, and similarly replace the Post Title with a relevant headline for the text you're about to write. Finally replace the text of the Gettysburg Address with your own text answering the following question, making sure you don't delete any of the html tags like <p> or </p>. 

 Looking at the class norms:
  - what's one that you think you can do pretty easily?
  - and what's one that you'll need to work hard at?

To view your changes, save the file in Atom then refresh the page in the browser.

Publish

Go back to your copy of the lab repo in github. Click the index.html file and put it in edit mode (by clicking the little pen icon). Go to Atom and copy the entire text of the index.html file you edited, then go back to github and paste it into the edit window. Click the green 'Commit changes' button.

Check your public webpage at http://your-username.github.io/FirstBlog.
