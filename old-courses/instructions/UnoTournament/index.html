Get Stuff

You'll need to:
  1. install jGRASP following the instructions here
  2. download this zip file
  3. extract it to a working location on your computer
  4. IMPORTANT: you must leave the zip file and go to the newly extracted folder in windows (!). you can trash the zip file now. you will get stuck below if you open or try to edit files in the zip file.
  5. double-click the `Uno.gpj` file (type "jGRASP project), which your computer should open in jGRASP

Definition of Done

You are done when:

  • Your code passes all the tests in `TestCaseProcessor`
  • meets the requirements of "For a B..." or "For an A..." in the grading section of the Uno Details document (linked above)
  • you have sent Mr. Smith an email (andrewsmith@wcsu.net) with your `keyboardcat_UnoPlayer.java` file as an attachment
