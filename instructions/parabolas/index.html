Getting started

  • set up github account
  • set up a p5js account
  • tinker with the template code to see what it does, and document noticings & wonderings

Instructions

  1. make a fresh copy of the code: load the template code (here), click File > Duplicate and optionally renaming the file
  2. claim a task: go to the tracking sheet for this project (gulian project here) and write your name in the 'claimed' column beside one of the filenames
    3. update the code: see noticings & wonderings for tips
  3. did you save the file?: click File > Save
  4. update the webpage: click File > Share and copy the Embed code. then go to the project webpage source file (link at the top of the tracking sheet), put it in edit mode, and paste your link in the first emply placeholder, then click Commit Changes (if there are any messages after this, let the scrum master know)
  5. mark the task as finished: go back to the tracking sheet and type ok in the done column

repeat as time allows
